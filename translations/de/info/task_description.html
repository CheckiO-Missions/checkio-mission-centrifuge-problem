Eine Zentrifuge hat n identische Schlitze, die jeweils groß genug sind, um ein Reagenzglas aufzunehmen. Um zu verhindern, dass die Zentrifuge wackelt, müssen k identische Röhrchen so in diese Schlitze gestellt werden, dass ihr gemeinsamer Schwerpunkt genau in der Mitte der Zentrifuge liegt. Dieses Problem wurde von einem anderen Video "Das Zentrifugenproblem" inspiriert, das Sie sich unten ansehen können. Alternativ können Sie sich auch den Beitrag von Matt Baker “The Balanced Centrifuge Problem” ansehen, dessen Augen schneller lesen als seine Ohren hören.

Es stellt sich also heraus, dass das Ausbalancieren von k Reagenzgläsern in n Steckplätzen nur dann möglich ist, wenn beide k, n-k als Summen von Primfaktoren von n ausgedrückt werden können, wobei die Wiederholung von Faktoren erlaubt ist. Wenn n zum Beispiel gleich 6 ist, dessen Primfaktoren 2 und 3 sind, kann die Zentrifuge 0, 2, 3, 4 (= 2 + 2) oder 6 (= 3 + 3) Reagenzgläser aufnehmen. Es gibt jedoch keine Möglichkeit, 1 oder 5 Reagenzgläser in sechs Fächern unterzubringen. Auch wenn 5 = 2 + 3 den ersten Teil der Regel erfüllt, gibt es keine Möglichkeit, den verbleibenden leeren Steckplatz auszugleichen, was trotz der kontraintuitiven Tatsache, dass leere Steckplätze nichts wiegen, erforderlich ist!

Diese Funktion muss die ausgewogene Konfiguration von k Reagenzgläsern nicht wirklich konstruieren, sondern lediglich feststellen, ob mindestens eine ausgewogene Konfiguration für die gegebene n, k existiert.
